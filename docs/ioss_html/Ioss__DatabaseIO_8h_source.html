424  /*!
425  * The overlay_count specifies the number of restart outputs
426  * which will be overlaid on top of the currently written
427  * step before advancing to the next step on the restart
428  * database.
429  *
430  * For example, if restarts are being output every 0.1
431  * seconds and the overlay count is specified as 2, then
432  * restart will write time 0.1 to step 1 of the database.
433  * It will then write 0.2 and 0.3 also to step 1. It will
434  * then increment the database step and write 0.4 to step 2;
435  * overlay 0.5 and 0.6 on step 2... At the end of the
436  * analysis, assuming it runs to completion, the database
437  * would have times 0.3, 0.6, 0.9, ... However, if there
438  * were a problem during the analysis, the last step on the
439  * database would contain an intermediate step.
440  *
441  * The cycle_count specifies the number of restart steps
442  * which will be written to the restart database before
443  * previously written steps are overwritten.
444  *
445  * For example, if the cycle count is 5 and restart is written every 0.1
446  * seconds, the restart system will write data at times 0.1, 0.2, 0.3,
447  * 0.4, 0.5 to the database.
448  *
449  * It will then overwrite the first step with data from time 0.6, the
450  * second with time 0.7. At time 0.8, the database would contain data at
451  * times 0.6, 0.7, 0.8, 0.4, 0.5. Note that time will not necessarily be
452  * monotonically increasing on a database that specifies the cycle
453  * count.
454  *
455  * The cycle count and overlay count can both be used at the same time
456  * also. The basic formula is:
457  *
458  * db_step = (((output_step - 1) / overlay) % cycle) + 1
459  *
460  * where "output_step" is the step that this would have been on the
461  * database in a normal write (1,2,3,....) and "db_step" is the step
462  * number that this will be written to.
463  *
464  * If you only want the last step available on the database,
465  * use set_cycle_count(1)
466  */
467  void set_cycle_count(int count) const { cycleCount = count; }
468  int get_cycle_count() const { return cycleCount; }
469  void set_overlay_count(int count) const { overlayCount = count; }
470  int get_overlay_count() const { return overlayCount; }
471  void set_file_per_state(bool yes_no) const { filePerState = yes_no; }
472  bool get_file_per_state() const { return filePerState; }
